# 自用模式
自用模式是一个特殊的运行模式，适用于用户仅自己使用 Bot 的场景。在自用模式下，Bot 将不设置群聊命令，并且某些命令会返回提示信息，告知用户已启用自用模式。此模式适用于只希望自用Bot的用户。
## 启用自用模式
要启用自用模式，请在 `config.yaml` 文件中找到 `misc` 部分，并将 `self_use` 字段设置为 `true`：
```yaml
misc:
  self_use: true
```
警告：启用自用模式后，请务必将你的用户 ID 添加到 `owner_id` 中，否则你将无法访问 Bot。示例如下：
```yaml
misc:
  self_use: true
  owner_id: 123456789
```
## 自用模式的效果
启用自用模式后，Bot将会有以下变化：
- Bot 将不会在群聊中设置命令，因此用户在群聊中与 Bot 交互时将看不到 Bot 的命令列表。
- 某些设计用于群聊使用的功能可能会被禁用或修改，以更适合个人使用。例如，`/usage` 命令将显示一条消息，指示在自用模式下没有使用限制。
- `/about` 命令将包含一条消息，指示已启用自用模式。
- 捐赠功能将被强制禁用，因为它不适用于自用模式。
