# Problems with the upgrade function
The upgrade function currently has the following known issues:
1. There is a compatibility issue between versions 1.8.x and 1.9.x.
2. There is a compatibility issue between versions 1.9.x and 1.10.x.
## Solution
If you are currently using version 1.8.x, it is recommended to manually upgrade to version 1.9.0 or higher for a better upgrade experience and to avoid issues during the upgrade process. 
If you are currently using version 1.9.x, it is recommended to manually upgrade to version 1.10.0 or higher for a better upgrade experience and to avoid issues during the upgrade process.
